Welcome to the Weather App! This application fetches real-time weather data from a weather API and displays various weather-related information for a selected location. The app dynamically changes its background images based on the current weather conditions, offering a visually engaging experience.

# Features

Real-Time Weather Data: Fetches the latest weather information, including temperature, humidity, wind speed, and more.

Dynamic Backgrounds: The app uses different images to reflect the current weather, such as sunny, rainy, snowy, cloudy, etc.

Location Search: Users can search for weather information by entering the name of a city or location.

Weather Forecast: Displays the weather forecast for the upcoming days.

User-Friendly Interface: Simple and intuitive design for a seamless user experience.


# Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: Fetch API to connect with the weather API

Weather API: OpenWeatherMap (or any other weather API you choose)
